\section*{Text movements}

\noindent
You can move texts around with the \verb"\advitransbox" \TeX~macro.

\noindent For instance,

\verb"\advitransbox{slide,from=right}{coming from the right}}"

\noindent moves the text {\em coming from the right} from the
right edge of the slide to its correct position into the
page. Variants \verb"right", \verb"left", \verb"top", and
\verb"bottom" of this macro are available.
\\
This is now being generalised to an open framework to describe
arbitrary paths for the box to follow in the animation.

\newpage

For example, the first effect is obtained via:\\
%
\verb"\advitransbox{path,steps=100,startx=500,starty=1000,stopx=10,stopy=10,genpath=line}"

Notice that, if start or stop is omitted, it is replaced by the actual
coordinates of the box in the dvi screen, so that the effect for the
\textcolor{yellow}{yellow} line below is obtained by writing:\\
\verb"\advitransbox{path,steps=100,startx=500,starty=1000,genpath=line}{path}"
{\Huge
\begin{itemize}
\item \textcolor{red}{From \advitransbox{path,steps=100,startx=500,starty=1000,stopx=10,stopy=10,genpath=line}{path}}
\item \textcolor{yellow}{From \advitransbox{path,steps=100,startx=500,starty=1000,genpath=line}{path}}
\item \textcolor{green}{From \advitransbox{slide,from=left}{left}}
\item \textcolor{blue}{From \advitransbox{slide,from=top}{top}}
\item \textcolor{cyan}{From \advitransbox{slide,from=bottom}{bottom}}
\end{itemize}
}
