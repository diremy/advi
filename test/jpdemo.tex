\subsection* {悪恥部 泥部位愛 の日本語表示について}
悪恥部 泥部位愛(以下『悪泥』)ではあなたが今御覧になっている通り、
p\TeX の日本語 DVI ファイルの表示が可能です。
日本語を表示するには以下の事を行っています。

\begin{itemize}
\item Camlimages の Freetype インターフェースを用いた
  日本語 true type フォントを使った日本語文字レンダリング
  (ですから Freetype が使えるようにコンパイルされた Camlimages 
  と一緒にリンクされていない場合は日本語表示できません。)
\item {\tt vftools} のソースから頂いたデータ
  を元にした括弧（「、」）や句読点などの位置補正
\end {itemize}

\subsection* {バグ}
いくつかバグがある。
\begin{itemize}
\item 日本語のベースラインが変。(資料を探しているところ)
\item Truetype フォントファイル名({\tt msmincho},{\tt msgothic})
  がハードコーデッドされている
  (でも、 VFlib のインターフェースなんか書きたくない。)
\item Truetype フォントは kpsewhich で見えるようにしておく事 
\end {itemize}
