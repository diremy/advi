\subsection* {悪恥部 泥部位愛 の日本語表示について}
悪恥部 泥部位愛(以下『悪泥』)ではあなたが今御覧になっている通り、
p\TeX の日本語 DVI ファイルの表示が可能です。
日本語を表示するために以下の事を行っています。

\begin{itemize}
\item Camlimages の Freetype インターフェースを通した
  日本語 true type フォントを使ったレンダリング
  (ですから Freetype が使えるようにコンパイルされた Camlimages 
  と一緒にリンクされていない場合は日本語表示できません。)
\item {\tt vftools} のソースから頂いたデータ
  を元にした括弧（「、」）や句読点などの位置補正
\end {itemize}

\subsection*{フォント設定}
いまさらVFlib のインターフェースなんか書きたくないので、
\begin{itemize}
\item \verb+$(ADVILIB)/jpfonts.conf+
\item もしくは \verb+~/.advi/jpfonts.conf+
\end{itemize}
ファイルに自前の\TeX{}フォント $\rightarrow$ TrueTypeフォントの
マップ情報を持っています。

\subsection* {バグ}

\end {itemize}
